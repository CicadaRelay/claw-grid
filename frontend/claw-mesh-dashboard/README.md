# Claw Mesh Dashboard

Mesh + MemoV 深度融合的可视化控制台

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 功能

- 🌐 Mesh 拓扑图（React Flow）
- ⏱️ MemoV 时间线（实时 WebSocket）
- 🐛 因果调试器
- 🔍 全局搜索（MemoV + Qdrant）
- ⏮️ 时光回滚

## 技术栈

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- React Flow
- Zustand
- Socket.io
